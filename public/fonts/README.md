# خط IBM Plex Sans Arabic

يرجى إضافة ملفات خط IBM Plex Sans Arabic في هذا المجلد:

**الأسماء المطلوبة:**
- `IBMPlexSansArabic-Regular.woff2` (مفضل)
- `IBMPlexSansArabic-Regular.woff`
- `IBMPlexSansArabic-Regular.ttf`
- `IBMPlexSansArabic-Bold.woff2` (مفضل)
- `IBMPlexSansArabic-Bold.woff`
- `IBMPlexSansArabic-Bold.ttf`

**أو بأسماء بديلة:**
- `IBM-Plex-Sans-Arabic-Regular.*`
- `IBM-Plex-Sans-Arabic-Bold.*`

**مصادر تحميل الخط:**
- GitHub: https://github.com/IBM/plex
- يمكنك تثبيت الحزمة: `npm install @ibm/plex`
